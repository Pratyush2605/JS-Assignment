JavaScript Array Methods 1.The JavaScript method toString() converts an array to a string of (comma separated) array values. eg:var fruits = ["Banana", "Orange"]; document.getElementById("demo").innerHTML = fruits.toString(); 2.The join() method also joins all array elements into a string. eg:var fruits = ["Banana", "Orange"]; document.getElementById("demo").innerHTML = fruits.join(" # "); 3.The shift() method removes the first array element and "shifts" all other elements to a lower index. 4.The splice() method can be used to add new items to an array. JavaScript String Methods 1.The lastIndexOf() method returns the index of the last occurrence of a specified text in a string. 2.slice() extracts a part of a string and returns the extracted part in a new string. 3.The replace() method replaces a specified value with another value in a string. 4.The trim() method removes whitespace from both sides of a string.
